Day Pillars - A Simple To-Do App

A modern and minimalist to-do list app designed to help you organize your daily tasks with ease. Built with Jetpack Compose, the app focuses on a clean, intuitive user experience.

Jetpack Compose Kotlin MVVM Room Database

Key Features:

  • Add, Edit, & Delete Tasks
  • Real-time search functionality
  • Filter & Sort options
  • Priority System (High, Medium, Low)
View on GitHub

How to Use the Gallery:

  • Click anywhere on the project card to open the full gallery
  • Use arrow keys (← →) to navigate between images
  • Swipe left/right on mobile devices
  • Click the indicators at the bottom to jump to specific images
  • Press Escape or click outside the modal to close
